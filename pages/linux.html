13. Create one logical volume named database and it should be on datastore volume group with size 50 extent(ext4).

(i) the datastore volume group extend should be 16MiB. (ii)mount the logical volume under mount point /mnt/database.

vgcreate -s 16 datastore /dev/vdb7
lvcreate -l -50 -n /dev/datastore/database
mkfs.ext4 /dev/datastore/database

17. The initial size of that logical volume database is 800MB. Make successfully reduce the size of logical volume 500MB without losing any data.

lvdisplay
LV size 800.00 MB // decrease to 500.00
Current LE 50

// In ext4 size can be increased or decreased but not in xfs.

U Umount
E e2fsck //extend to file system check // defragmentation
R resize2fs // resize to file system
L lvreduce
// Mounted file system can not be decreased.


e2fsk -f /dev/datastore/database
resize2fs /dev/datastore/database 500M
lvreduce -L 500M /dev/datastore/database
