LDAP :

Ldap server name is : classroom.example.com

Ldap user authentication-
In this location lot of users are ther from 0 to 20.
Light weight directory access protocol.

Your home directory is available at classroom.example.com:/home/guests

Authenticating Ldap user : 80 marks

yum install authconfig-gtk autofs sssd

sssd : system securuty service deamon
Authentication and authorization
password field is authentication.

#usermod -s /sbin/nologin rakesh

If there is nologin permission, it's not authorized.

#autoconfig-tui

Ldap users is available in the location:
/home/guests/ldapuser0
Mark with the astric containing Ldap.
Next Ldap settings
Use Tls : transport layer security

server : classroom.example.com
DN : Distinguishing name : dc=example,dc=com
warning message is displayed
authority certificate
authconfig-gtk //just enter
download CA certificatio and provide certificate url path.

http://classroom.example.com/pub/example-ca.crt
apply
su - ldapuse0
systemctl restart sssd
systemctl enable sssd
Now bash will come
//output
cannot change directory /home/guests

Auto mounting the home directory
vim /etc/auto.master

press i
+auto.master
/home/guests  /etc/auto.misc

vim /etc/auto.misc
shift + g
ldapuser0 -rw,soft,intr classroom.example.com:/home/guests/ldapuser0

What is soft and hard mount ?
Soft mount checks every 5 minutes,the server. 'Intr' means interrupt.
While the hard mount have to do it manually everytime.

systemctl restart autofs
systemctl enable autofs


then output will be like :
